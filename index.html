      What is a game loop?
      
          
      A function that runs about 60 times per second. Each frame it: (1) reads input, (2) updates the world (move things, check collisions), (3) draws the screen. Look for function loop() and requestAnimationFrame(loop).
      

          
      What is a sprite?
      
          
      A 2D image used to draw a character or item. Your uploaded picture becomes the player sprite. Without an upload, the game draws a colored rectangle in its place.
      

          
      Collision detection
      
          
      Checking if two shapes overlap. The simplest test is the bounding box (AABB) helper named hits(a, b) — it returns true when two rectangles overlap on both X and Y.
      

          
      Score and timer
      
          
      The score grows when you do something good (dodge, hit a target, collect an item). The timer counts seconds since the game started. Both are updated inside the loop and shown in the HUD.
